# 🧱 Brick by Brick – The Evolution of the LEGO Universe

An interactive Power BI dashboard that explores the LEGO ecosystem using the Rebrickable dataset.

---

# 📌 Project Overview

This project analyzes LEGO sets, themes, colors, and parts using Microsoft Power BI. The dashboard helps users explore LEGO trends through interactive visualizations and storytelling.

---

# 🎯 Objectives

- Analyze LEGO growth over the years
- Identify the most popular LEGO themes
- Explore the largest LEGO sets
- Compare number of parts across sets
- Create an interactive storytelling dashboard

---

# 🛠️ Tools & Technologies

- Microsoft Power BI
- Power Query
- DAX
- Rebrickable LEGO Dataset
- Microsoft Excel
